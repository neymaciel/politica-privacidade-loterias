Política de Privacidade

Última atualização: 06 de junho de 2026

O aplicativo Loterias Caixa foi desenvolvido com o objetivo de fornecer informações, resultados, estatísticas e consultas relacionadas às loterias administradas pela Caixa Econômica Federal.

Coleta de Informações

O aplicativo não coleta, armazena nem compartilha informações pessoais dos usuários, tais como nome, CPF, endereço ou dados bancários.

Permissões do Aplicativo

O aplicativo pode utilizar acesso à internet exclusivamente para obter resultados, concursos, estatísticas e demais informações públicas relacionadas às loterias.

Dados de Uso

Alguns dados técnicos podem ser coletados automaticamente pelos serviços utilizados para funcionamento do aplicativo, como endereço IP, tipo de dispositivo e versão do sistema operacional.

Publicidade

Caso o aplicativo utilize serviços de publicidade, parceiros terceiros poderão coletar informações conforme suas próprias políticas de privacidade.

Compartilhamento de Dados

Nenhuma informação pessoal é vendida, compartilhada ou transferida para terceiros.

Segurança

Empregamos medidas razoáveis para proteger as informações processadas pelo aplicativo.

Links para Terceiros

O aplicativo pode conter links para sites externos. Não nos responsabilizamos pelas políticas de privacidade ou conteúdo desses sites.

Alterações nesta Política

Esta Política de Privacidade poderá ser atualizada periodicamente. Recomendamos que os usuários consultem esta página regularmente.
